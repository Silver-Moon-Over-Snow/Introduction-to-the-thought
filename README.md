# 习概简答题刷题

一个用于复习“习概”简答题的本地网页小工具。打开 `index.html` 即可使用，不需要安装依赖。

## 功能

- 按题目逐题练习简答题
- 先判断答案要点数，再默写答案
- 自动核对答案，记录错题
- 支持收藏题目、斩题、备注
- 支持错题本、收藏本、已答题、已斩题视图
- 支持浅色/深色主题
- 支持导出和导入刷题进度

## 使用方法

直接用浏览器打开：

```text
index.html
```

如果浏览器限制本地文件功能，也可以在当前目录启动一个简单的本地服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 数据说明

做题记录默认保存在当前浏览器的 `localStorage` 中，不会自动上传到 GitHub。

页面里的“导出进度”会生成本地 JSON 文件，用于备份或迁移记录；这类文件已被 `.gitignore` 忽略。

## 文件结构

- `index.html`：刷题网页主程序
- `习概_主题问题答案提取.md`：复习资料整理
- `check_commas.py`：题库数组逗号检查脚本
- `祥云.png` / `祥云2.png` / `祥云3.png`：页面装饰素材
